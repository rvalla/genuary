I had never worked with flow fields so I have a lot of troubles to find how to use them on my own way. I learnt that it isn't easy to create a good field. I am happy with the final results though. This images were made with a flow field but they are not up to the standard of Tyler Hobbs' fidenza.

Flow fields

Using a flow field is not so difficult and is really fantastic to draw organic curves. You simply create a matrix of angles on top of your canvas which determine the direction of the lines. So, every line that passes trough a point (x,y) has the same direction.

Jumps

I finally found my own way to use flow fields adapting an idea which I use a lot. Basicaly I make the computer draw following the field but with random jumps. Sometimes there are tiny circles, sometimes only straight lines...
