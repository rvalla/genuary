I don't make a lot of visual loops really. I like them though. This year I have not much time this month (I want to improve my chess). So I decided to start with simple harmonic signals and a canvas its width is not a multiple of the signal period.

A pretty curve

An harmonic signal draws a curve that seems irregular when you don't see several cycles. If we superimpose different parts of the curve by changing the color we get interesting results.
